## Truck Haven (Admiral Trucker) 🚛
Truck Haven is a professional frontend web application built for Admiral Trucker Limited, a premier logistics and haulage service provider based in Nigeria. The site serves as a digital storefront showcasing nationwide equipment shipment, general goods haulage, and specialized leasing services.

# Live Demo
## Features
## Interactive Navigation: 
Seamless access to Home, About, Services, Tracking, and Career pages.

## Service Showcase: 
Detailed sections for Road Haulage, Warehousing, and Equipment Leasing.

## Geographic Coverage: 
A dedicated section highlighting nationwide delivery capabilities across Nigeria.

## Responsive Design: 
Optimized for various screen sizes, ensuring a smooth user experience on mobile, tablet, and desktop.

## Dynamic UI: 
Clean layout with modern CSS transitions and high-quality imagery.

## News & Events: 
A section to update users on company milestones and awards.

## Built With
This project was developed using a modern frontend stack to ensure speed and performance:

## HTML5: Semantic markup for better SEO and accessibility.

## CSS3: Custom styling including Flexbox/Grid for layout and responsive media queries.

## JavaScript: For interactive components and navigation logic.

## Github: Used for continuous deployment and hosting.
